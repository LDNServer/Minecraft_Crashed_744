A cursed Minecraft version made with LDT  
Minecraft 1.1 (Pending)

## What is this?
- This is a mod-based cursed Minecraft that works on **Java 8** using **MultiMC** or **PrismLauncher**.  
- You need a Minecraft (Microsoft) account to play (paid). If you already have an account, you can use it.  
- This is **not** a Bedrock Edition (mobile, Switch) addon.

## How to Play

### Client
**Requirements:** Java 8 (Zulu recommended)  
[Download Zulu Java 8](https://www.azul.com/downloads/?version=java-8-lts&os=windows&architecture=x86-64-bit&package=jdk#zulu)  

1. Download the client zip file:  
[Crashed_744.zip](https://github.com/YonCursedMC/Minecraft_Crashed_744/releases) (choose the top one).  

2. Install MultiMC or PrismLauncher:  
- [MultiMC](https://multimc.org/#Download)  
- [PrismLauncher](https://prismlauncher.org)  

   After installation, log in with your Minecraft (Microsoft) account, then **drag & drop** the zip file downloaded in step 1.  

3. Configure the installed instance to use the Zulu Java 8 you installed. Adjust memory allocation if needed, then click **Launch**.  

4. Enjoy!  

### Server
1. Download the Minecraft 1.1 server jar:  
[1.1.jar](https://files.betacraft.uk/server-archive/release/1.1/1.1.jar)  

2. Download the Crashed_744 server zip:  
[Crashed_744_server.zip](https://github.com/Minecraft_Crashed_744/Minecraft_666_Edition/releases)  

3. Extract `1.1.jar` using 7-zip, then overwrite its contents with `Crashed_744_server.zip`.  

4. Repackage the jar and launch. If it shows **Minecraft Crashed 744**, youâ€™re ready.  

> **Note:** Server behavior differs from the client. Many bugs exist, so play with patience!
